Login Info

Author

Alistair Kearney

Version

1.5 released on 10 January 2010

Description

This is an event that displays basic login details (such as their real name, username and author type) if the person viewing the site have been authenticated by logging in to Symphony. It is useful if you want to do something special with the site if the person viewing it is an authenticated member.
